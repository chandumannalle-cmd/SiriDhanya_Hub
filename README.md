# 🌾 SiriDhanyaHub

## 📌 Project Overview

SiriDhanyaHub is an Android application developed to promote millet awareness, healthy food habits, and direct farmer-to-consumer interaction. The app helps users explore millet recipes, health benefits, market details, and purchase millet products directly from farmers.

The application provides a simple and user-friendly platform that supports healthy living and sustainable agriculture.

---

# ✨ Features

- 🏠 Home Dashboard
- 📊 Mandi Watch for market information
- 🍲 Millet Recipe Lab
- ❤️ Health Benefits Section
- 🛒 Direct Farmer-to-Consumer Purchase
- 📱 Simple and Responsive UI

# 📂 Project Modules

## 🏠 Home Module
Provides navigation to all sections of the application.

## 📊 Mandi Watch
Displays millet market-related information and pricing details.

## 🍲 Recipe Lab
Shows healthy millet recipes with ingredients and preparation steps.

## ❤️ Health Benefits
Provides nutritional and health-related information about millets.

## 🛒 Direct Buy Module
Allows consumers to connect directly with farmers.
